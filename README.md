# ph-holiday
Java library containing holiday configuration.

# News and noteworthy

## Maven usage
Add the following to your pom.xml to use this artifact:
```
<dependency>
  <groupId>com.helger</groupId>
  <artifactId>ph-holiday</artifactId>
  <version>5.0.1</version>
</dependency>
```
